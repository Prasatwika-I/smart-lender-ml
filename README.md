# 🏦 Smart Lender ML

> An AI-powered Loan Approval Prediction System built using Machine Learning and Flask.

---

## 📖 Project Overview

Smart Lender ML is a Machine Learning-based web application developed to predict the creditworthiness of loan applicants. The system helps banks and financial institutions make faster, data-driven loan approval decisions by analyzing applicant information and predicting whether a loan should be approved.

The application uses multiple Machine Learning algorithms, compares their performance, and deploys the best-performing model through a Flask web application.

---

## ✨ Features

- Loan approval prediction
- Credit risk assessment
- User-friendly web interface
- Machine Learning-based prediction
- High prediction accuracy
- Fast and reliable decision making

---

## 🤖 Machine Learning Algorithms

The project implements and evaluates multiple Machine Learning algorithms for loan approval prediction.

| Algorithm | Description |
|-----------|-------------|
| Decision Tree | Tree-based classification algorithm |
| Random Forest | Ensemble learning algorithm |
| K-Nearest Neighbors (KNN) | Distance-based classification |
| XGBoost | Gradient Boosting algorithm |

### Best Performing Model

**XGBoost**

- Training Accuracy: **94.7%**
- Testing Accuracy: **81.1%**

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Prasatwika-I/smart-lender-ml.git
```

### Navigate to the Project

```bash
cd smart-lender-ml
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

## ▶️ Usage

1. Launch the Flask application.
