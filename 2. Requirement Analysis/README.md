# Requirement Analysis

## Objective
This folder contains the requirement analysis documents for the Smart Lender ML project.

## Contents

### Functional Requirements
- User can enter loan applicant details.
- System predicts loan approval status.
- System displays prediction results.
- Admin can manage the application.

### Non-Functional Requirements
- High prediction accuracy.
- Fast response time.
- Secure handling of user data.
- User-friendly interface.

### Software Requirements
- Python
- Flask
- Jupyter Notebook
- IBM Cloud
- Visual Studio Code

### Hardware Requirements
- Processor: Intel Core i3 or above
- RAM: 4 GB or above
- Storage: 10 GB free space

### Use Cases
- Loan Approval Prediction
- Applicant Data Analysis
- Credit Risk Assessment
